Analisis Arsitektur Proyek: Absen WFA

Aplikasi Progressive Web App (PWA) untuk sistem absensi mandiri yang terintegrasi penuh dengan Microsoft 365.

Microsoft Graph API SharePoint Lists MSAL.js PWA

📂 Struktur & Ringkasan Skrip

Komponen Fungsi Utama
sw.js Service Worker: Caching offline & performa akses cepat.
js/config.js Pusat konfigurasi ID Azure, SharePoint, & Jam Kerja.
js/auth.js Manajemen login/logout & Token Microsoft Identity.
js/graph.js Backend-as-a-Service: Komunikasi ke SharePoint & Outlook.
js/app.js Main Controller: Navigasi (Calendar, Absen, Request).
js/utils.js Helper: Format tanggal, durasi, & indikator UI.

🔄 Alur Proses & Persetujuan (Implicit Approval)

Aplikasi menggunakan alur Persetujuan Implisit untuk mempercepat proses kehadiran:

  • Auto-Approved: Setiap pengajuan baru dari karyawan otomatis disimpan dengan status Approved di SharePoint, sehingga karyawan dapat langsung melakukan presensi pada hari H tanpa terhambat verifikasi manual.
  • Notifikasi Email Atasan: Sistem mengirimkan email notifikasi ke atasan (CC ke HRD & MIS) segera setelah pengajuan dibuat.
  • Opsi Pembatalan Tunggal: Di dalam email notifikasi, atasan hanya diberikan tautan tunggal untuk membatalkan/menolak pengajuan (Reject/Batalkan), tanpa perlu memproses persetujuan (Approve) secara aktif.
  • Proses Penolakan: Jika atasan membatalkan, status di SharePoint diubah menjadi Rejected (mengunci tombol absen untuk hari tersebut), dan email pembatalan dikirim kembali ke karyawan dengan catatan/alasan penolakan yang bersifat opsional.

🛠️ Rekomendasi Peningkatan

Optimasi Query (Server-side Filtering) Menerapkan OData $filter pada Microsoft Graph API untuk mengurangi beban memori browser saat data karyawan/absensi bertambah ribuan baris.
Fitur Offline Resilience Menggunakan IndexedDB atau localStorage untuk menyimpan antrean absen saat internet mati, lalu mengirimnya otomatis saat online.
Verifikasi Geolokasi Menambahkan pencatatan koordinat GPS saat tombol absen ditekan untuk meningkatkan integritas data kehadiran.
Modularitas & Scalability Memecah app.js menjadi modul-modul kecil (misal: auth-controller.js, calendar-view.js) untuk mempermudah kolaborasi tim.
Environment Variable Security Memindahkan kredensial teknis sensitif ke proses build/deployment untuk meminimalisir eksposur informasi infrastruktur.
